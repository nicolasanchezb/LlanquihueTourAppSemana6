# LlanquihueTourApp

## Objetivo de esta semana

Implementar una jerarquía de clases utilizando herencia simple para representar los distintos servicios turísticos ofrecidos por la agencia Llanquihue Tour.

## Clases creadas

### model

- ServicioTuristico
- RutaGastronomica
- PaseoLacustre
- ExcursionCultural

### data

- GestorServicios

### ui

- Main

## Conceptos aplicados

- Herencia simple
- Sobrescritura del método toString()
- Uso de super()
- Organización por paquetes

## Cómo ejecutar

1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar la clase `Main` ubicada en el paquete `ui`.
3. El programa mostrará por consola dos rutas gastronómicas, dos paseos lacustres y dos excursiones culturales.
